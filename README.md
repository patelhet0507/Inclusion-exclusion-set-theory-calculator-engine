# Set Theory Calculator

A modern web application for calculating set operations using the Inclusion-Exclusion Principle.

## Features

- Calculate union of 2 or 3 sets
- Calculate intersection of two sets
- Calculate difference of two sets
- Responsive design with glassmorphism UI
- Real-time calculation with AJAX
- Dynamic form fields based on operation

## Installation

1. Clone or download the repository
2. Navigate to the project directory
3. Create a virtual environment: `python -m venv .venv`
4. Activate the virtual environment: `.venv\Scripts\activate` (Windows)
5. Install dependencies: `pip install -r requirements.txt`

## Running the Application

Run the Flask app:

```bash
python app.py
```

Open your browser and go to `http://127.0.0.1:5000/`

## Usage

1. Select the operation you want to perform
2. Enter the required set sizes and intersections
3. Click "Calculate" to see the result

## Technologies Used

- Flask (Python web framework)
- HTML5, CSS3, JavaScript
- AJAX for dynamic updates
